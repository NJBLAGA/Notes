# **VSCODE Commands**

## Documentation

- [Visual Studio Code Shortcuts](https://code.visualstudio.com/shortcuts/keyboard-shortcuts-windows.pdf)
- [Keyboard Shortcuts](https://code.visualstudio.com/shortcuts/keyboard-shortcuts-windows.pdf)

## **Basics**

`Control + ,` User Settings

`Control + Shift + p` Show Command Palette

`Control + p` Quick Open

`Leader + p` Format

`Leader + w` Save File

`Leader + c` Comment Line in V Mode

## N**avigation**

`Alt + h` Focus Tree

`Alt + j` Focus Terminal

`Alt + k` Focus Editor

`Alt + l` Focus Next Group

`Shift + l` Next Buffer

`Shift + h` Previous buffer

## Panes

`Leader + s` Spilt

`Leader + v` Vertical Split

Panel Navigation

`Leader + h` Focus Left Pane

`Leader + l` Focus Right Pane

`Leader + j` Focus Below Pane

`Leader + k` Focus Above Pane

## File Tree

`Control + n` Toggle Sidebar

`Control + c` Create New File

`Control + f` Create New Folder

`Control + d` Delete File/Folder

`Shift + r` Rename File/Folder

`Shift + c` Copy File/Name

`Shift + v` Paste File/Name

## **Terminal**

`Shift + p` Focus Problems

`Control + j` Toggle Panel

`Shift + p` Focus Problems

`Shift + t` Focus Terminal

## **Debug**

`Control + Shift + d` Toggle Debug

`F9` Toggle Break-point

`F5` Start/Continue

`Shift + F5` Stop

`F11 / Shift + F11` Step into/out

`F10` Step over

`Control + K Control + l` Show hover

## **Extensions**

`Control + Shift + x` Toggle Extensions

`Control + Shift + r` Toggle Thunder Client

`Control + Alt + v` Open Vim Cheat-Sheet

`Alt + q` Start Live Server

`Control + Shift + m` Open Markdown Preview
