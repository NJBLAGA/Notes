# Awesome Window Manager Commands

## **Basics**

`Windows` Mod Key

`Modkey + w` Show Main Menu

`Modkey + s` Show Commands

`Modkey + Control + r` Restart Awesome

`Modkey + q` Close Window

`Modkey + Shift + q` Quit Awesome

`Modkey + f` Toggle Full Screen

`Modkey + Control + Space` Activate/Deactivate Floating

`Modkey + Click + Drag` Drag Window

## Launching **Basics**

`Modkey + Return` Launch Terminal

`Modkey + Space` Launch Fuzzy Finder

`Modkey + b` Launch Chrome

`Modkey + n` Launch Nautilus

`Modkey + z` Launch Flame Shot

## **Navigation Basics**

`Modkey + Shift + [Workspace Number]` Move Window to Workspace

`Modkey + j` Focus Next Window

`Modkey + k` Focus Previous Window

`Modkey + h` Focus Previous Workspace

`Modkey + l` Focus Next Workspace

`Modkey + o` Move Window To Next Screen

`Modkey + Shift + j` Switch To Master

`Modkey + Shift + k` Switch To Stack

`Modkey + Control + j` Switch To Next Screen

`Modkey + Control + k` Switch To Previous Screen

## Resizing **Basics**

`Modkey + m` Minimise | Maximise

`Modkey + Shift + -` Increase Master Width Factor

`Modkey + Shift + =` Decrease Master Width Factor

`Modkey + Shift + h` Increase The number of Master Clients

`Modkey + Shift + l` Decrease The Number of Master Clients
